# Fungsi Terbilang Rupiah

Fungsi **`terbilang`** adalah utilitas sederhana untuk mengubah angka menjadi teks dalam bahasa Indonesia, khususnya untuk nominal **Rupiah**.  
Contoh: `12500` â†’ `"dua belas ribu lima ratus rupiah"`

---

## Fitur
- Mengubah angka menjadi teks Rupiah dalam bahasa Indonesia.
- Mendukung angka kecil hingga besar (jutaan, milyaran, trilyunan).
- Mudah di tambah, tidak perlu menulis kode yang rumit/nested if, cukup tambahkan opsi di switchnya untuk pengucapan berikutnya.

---

## Instalasi
Karena ini hanya fungsi sederhana, cukup salin kode `terbilang` ke dalam project Anda.  
Tidak ada dependency tambahan.

---

## Cara Penggunaan

Jalankan program dan pilih berapa kali coba, dan masukkan angka di tiap percobaan.
